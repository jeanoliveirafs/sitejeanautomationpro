# 🚀 Jean Automation Pro — Portfolio & Agency Website

> **EN** | [🇧🇷 Versão em Português abaixo](#-sobre-o-projeto-português)

Official portfolio and landing page for Jean Automation Pro — a freelance automation and web development service. Showcases projects, services, and includes a contact form integrated with WhatsApp.

🌐 **Live Site:** [sitejeanautomationpro.vercel.app](https://sitejeanautomationpro.vercel.app)

---

## ✨ Features

- 🎨 **Modern Landing Page** — Clean, professional design for client acquisition
- 📬 **WhatsApp Contact** — Lead capture form integrated with WhatsApp
- 🗂️ **Portfolio Section** — Showcase of projects and case studies
- 💼 **Services Section** — Clear presentation of offered services
- 📱 **Fully Responsive** — Pixel-perfect on all screen sizes
- ⚡ **Fast Performance** — Optimized with Vite build system

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React, TypeScript, Vite |
| UI | TailwindCSS, Shadcn/UI |
| API | Node.js serverless functions |
| Deploy | Vercel |

---

## 🚀 Getting Started

```bash
git clone https://github.com/jeanoliveirafs/sitejeanautomationpro.git
cd sitejeanautomationpro
npm install
npm run dev
```
